# 🛒 GroceryMart React Native App

Welcome to **GroceryMart**, a modern, fast, and beautifully animated React Native application. This app was built using Expo Router to provide a seamless, app-like navigation experience and features a modern, clean UI tailored for a premium grocery shopping experience.

---

## ✨ Key Features

- **📍 Automatic Location Tracking:** 
  Utilizes `expo-location` and reverse geocoding to instantly detect and display the user's current city and street for delivery.
- **🚀 Expo Router Navigation:** 
  Modern, file-based routing mechanism eliminating the need for complex `NavigationContainer` setups. Routes are automatically generated based on the folder structure.
- **🛍️ Dynamic Category Browsing:** 
  A comprehensive categorization system allowing users to browse through Fresh Fruits, Vegetables, Dairy & Eggs, Meat & Poultry, and Bakery items.
- **🎨 Modern UI/UX:** 
  Features premium staggered "pop-in" animations, clean glassmorphism-inspired elements, and a sleek layout that rivals top-tier production apps.
- **🔍 Advanced Product Listing:** 
  Includes a simulated website-like product page with active filtering chips, search functionality, and detailed product cards with ratings.

---

## 🛠️ Tech Stack & Prerequisites

This project is built with a modern React Native stack. Before running the project, ensure you have the following installed:

- **[Node.js](https://nodejs.org/)** (v18 or newer recommended)
- **[Expo CLI](https://docs.expo.dev/)**
- **Expo Go App** (installed on your iOS or Android physical device)
